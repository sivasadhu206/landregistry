# 🏛️ LedgeLand Registry

A Blockchain-Based Land Registration System built on Ethereum Sepolia Testnet.

LedgeLand Registry is a decentralized application (DApp) that enables secure, transparent, and tamper-proof land record management using blockchain technology.

---

## 🚀 Project Overview

Traditional land registries rely on paper-based records, which are vulnerable to:

- Fraud
- Document tampering
- Data loss
- Corruption
- Lack of transparency

LedgeLand solves these problems by storing land ownership data on the Ethereum blockchain and linking property deeds via IPFS.

---

## 🔐 Key Features

### 👑 Admin (Registrar) Control
- Only the authorized registrar can register new land
- Smart contract enforces role-based access control

### 📥 Land Registration
- Upload land records using Excel file
- Automatically registers data on blockchain
- Each land gets a Unique Land ID

### 📜 IPFS Digital Deeds
- Property deed stored on IPFS (via Pinata)
- CID stored on blockchain
- Ensures document integrity

### 🔍 Public Land Verification
- Anyone can search using Land ID
- View:
  - Current Owner
  - Location
  - Ownership History
  - Deed Link

### 🔄 Ownership Transfer
- Secure blockchain-based ownership transfer
- Full chain of custody maintained
